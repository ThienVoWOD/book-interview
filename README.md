## Setup local env

#### 1. Use docker compose to setup ES container
```
docker-compose up -d
```
#### 2. Install package
- Use npm:
```
npm install
```
- Use yarn:
```
yarn install
```
#### 3. Change `.env.example` file to `.env`
#### 4. Start Server
- Use npm:
```
npm start
```
- Use yarn:
```
yarn start
```
#### 5. Run test
- Use npm:
```
npm test
```
- Use yarn:
```
yarn test
```
